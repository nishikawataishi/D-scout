# プロジェクト憲章: D.scout (Doshisha Scout)

## 1. プロジェクトの背景・目的
現在、大学生（特に同志社・同女大生）にとって、数多くのサークルやゼミから自分に合ったものを見つけることは「選ぶ疲れ」を伴う作業となっています。一方、団体側も適切な学生へのアプローチに課題を抱えています。
本プロジェクトは、「選ぶ疲れ」を「選ばれる喜び」へと転換し、学生と団体の最適なマッチングを実現するスカウトプラットフォームを提供することを目的とします。

## 2. 目標 (Objective)
*   **同志社・同女大生限定のクローズド環境の構築**: 大学ドメインメールによる厳格な認証。
*   **洗練されたUXの提供**: 「ONE CAREER」をベンチマークとした、直感的で美しいUI。
*   **スカウト機能の最大化**: 団体からの能動的なアプローチを中心としたマッチング体験。

## 3. スコープ (Scope)
*   **対象**: 同志社大学、同志社女子大学の学生および公認/準公認/有志団体。
*   **主要機能**: 
    *   大学ドメイン限定認証
    *   団体検索・チップフィルタリング
    *   スカウト通知・承認・SNS連携
    *   キャンパス別カラーリングによるイベント管理
    *   インタラクティブな興味関心タグ管理
*   **対象外**: 他大学学生の利用、一般企業の求人スカウト。

## 4. 制約条件・前提条件
*   **制約**: 
    *   メールアドレスは `@doshisha.ac.jp` または `@mail.doshisha.ac.jp` に限定。
    *   スマホサイズ（最大幅 448px: max-w-md）への最適化。
*   **前提**:
    *   デザインは白基調、アクセントカラーはインディゴブルー（indigo-600）を使用。

## 5. 主要なステークホルダー
*   **プロジェクトオーナー**: 西川様（ユーザー）
*   **開発チーム**: Antigravity (AIエージェント)
*   **エンドユーザー**: 同志社大学・同志社女子大学の学生
*   **団体ユーザー**: 参加するサークル・ゼミの代表者・広報担当者
